# 🎓 隧三智能课堂点名系统

一个基于Python和Tkinter开发的精美课堂点名工具，支持随机抽取学生编号，界面美观，操作简单。

![Python](https://img.shields.io/badge/Python-3.6+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)
![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20macOS%20%7C%20Linux-lightgrey.svg)

## ✨ 功能特色

### 🎯 核心功能
- **随机点名**: 从1-45号学生中随机抽取
- **动态效果**: 点名过程中数字颜色动态变化
- **历史记录**: 自动保存每次点名结果和时间戳
- **统计信息**: 显示当日点名次数统计

### 🎨 界面特色
- **现代化UI**: 采用美观的配色方案和布局
- **响应式设计**: 支持窗口大小调整
- **中文优化**: 完美支持中文显示
- **视觉反馈**: 通过颜色变化提供操作反馈

### 📊 数据管理
- **记录导出**: 支持历史记录导出功能
- **数据清空**: 一键清空历史记录
- **自动保存**: 点名记录自动保存
